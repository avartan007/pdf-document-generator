About the Engine

The PDF Report Generator was designed as a lightweight alternative to manual document sketching. It bridges the gap between raw research and formal presentation by leveraging a hybrid Node.js and Python architecture.

While many tools rely on heavy LLM calls, this generator uses a custom template-driven engine focused on document structure and thematic coherence. It handles text-wrapping, typography, and layout calculation programmatically using `pdf-lib`.
